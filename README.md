# vue-template
Built from the vue-cli. Modified to include bootstrap-vue.

## Requirements
**Node** 10.12.x
**NPM** 6.4.x

Easiest way is to use NVM.
```
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
nvm install 10.12
nvm use 10.12
```

## Project setup
```
npm install
```
