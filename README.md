# Portfolio 2019

Resume supplement to showcase my knowledge and development life cycle.

## Requirements

**Node** 10.16.x
**NPM** 6.9.x

Easiest way is to use NVM.

```bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
nvm install 10.16
nvm use 10.16
```
